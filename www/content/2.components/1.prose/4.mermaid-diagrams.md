## Usage

::stack
  ::div{class="p-4"}
  ```mermaid
  sequenceDiagram
    Client->>Server: Send Request
    Server->>Database: Query Data
    Database-->>Server: Return Data
    Server-->>Client: Return Response
  ```
  ::
